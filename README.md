Basic onboarding screen written with SwiftUI.

## Features

- [x] Present pages.
- [x] Next, Skip, and Back buttons.

## Technology and API

- SwiftUI
- AppStorage
- Responsive next button that shows progress

## Requirements

- Xcode > 14.0
- Swift > 5.0

## How to run
1. Clone the repo
2. Open OnboardingScreen.xcodeproj
3. Select a simualtor and click on run (CMD+R)
